Bussines Inteligence

Informações para o processo de tomada de decisão.

Multiplas fontes de dados

Por vezes as fontes de dados são as mais variadas possiveis, um BI poderoso tem a capacidade de obter e arrumar todas essas informações para apoiar o processo de tomada de decisão das empresas.

Painéis interativos

Paineis (dashboads) que permitem a aplicação de filtros, drilldown, slice and dice e outras formas de montar cenários dos dados analisados.

Multi plataforma

Microsoft Analisys Services, Power BI, Pentaho, R e Python são algumas plataformas de desenvolvimento de painéis e analises de dados.

Segmentação de Informação com regras para usuários

Possibilidade de filtrar os dados de acordo com o perfil de usuários de forma automática.

FAST BI ou Tradicional

Dados com volume muito alto precisam do desenvolvimento de um datawarehouse (armazem de dados) para um pré processamento. Outras análises precisam ser em tempo real. Casa situação será analisar para propor a melhor alternativa.

Projetos de BIG DATA

Tratamento de grandes volumes de dados e análises em tempo real de informações.
